HTML

HTML: coding language that works on page structure: Text elements, Image elements, Header elements, List elements

  • The "< head >" element contains information, or metadata, that helps the browser to render the page correctly. Elements included within the are not visible when a user looks at a webpage.
  • The "< body >"" element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.

CSS

  • Cascading Style Sheets (CSS): a programming language used by developers to define how webpage content is presented to users (aesthetics)
  • Inline CSS: apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. They’re useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
